# Assets Needed

Add the following assets to complete the branding:

## Required

- `favicon.svg` - SVG favicon (primary)
- `favicon.ico` - ICO fallback for older browsers
- `apple-touch-icon.png` - 180x180px for iOS home screen

## Recommended

- `images/logo.svg` - Full logo/wordmark
- `images/og-image.png` - 1200x630px for social media previews

## Brand Colors (from CSS)

- Primary Blue: hsl(212, 79%, 46%) / #1976D2
- Dark Blue: hsl(212, 79%, 29%)
- Accent Cyan: hsl(193, 69%, 62%)
- Highlight Yellow: hsl(48, 100%, 53%)
